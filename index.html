POLYA'S METHOD


It's a strategy created for solving problems with the following four steps:

  • UNDERSTAND THE PROBLEM

    • Do you understand everything from the problem?
    • Can you replant the problem in your own words?
    • Do you distinguish the data?
    • Do you know what is the goal?
    • s there enought information?
    • Is there strange information?
    • Do i resolve something similar before?

  • PLAN CONFIGURATION

    • Trial and error
    • Use a variable
    • Find a patron
    • Make a list
    • Resolve a simple similar problem
    • Make a figure
    • Use direct reasoning
    • Use the properties from the numbers
    • Resolve an equivalent problem
    • Work backwards
    • Use cases
    • Resolve an ecuation
    • Search a formule
    • Take a model and use this one
    • Use dimensional analisis
    • Identify sub-goals
    • Use coords
    • Use simetry

  • PLAN EXECUTE

    Implement the strategies and don't be scared of start again

  • LOOK BACKWARDS

    Is my solution correct? It satisfies the stablecid?
    Is there an easiest solution?
    Do you know how to extend your solution to a general case?

Problema 3: Con el propósito de obtener fondos para la actividad de la alerta temprana contra el cáncer de mama doña Claudia espera vender 27 cajas de estampas conmemorativas de la actividad. Tiene cajas que contiene Ocho estampas. Las cajas con estampas rosadas contienen nueve estampas. Si se adquirieron cajas con un total de 230 estampas. ¿Cuántas cajas de cada tipo se tienen?

FASE 1: Para poder obtener la cantidad de cajas de cada tipo que se tienen, debemos de ver que datos se tienen, y como podemos ver, tenemos un total de 27 cajas divididas en dos tipos:
Las que contienen estampas doradas, con ocho estampas
Las que contienen estampas rosadas, con nueve estampas
FASE 2: Esta información es suficiente para poder realizar nuestro ejercicio con la siguiente ecuacion: 230 = (9 * a) + (8 * b),
a + b = 27

POSIBLE SOLUCION

230 = (9 * 6) + (8 * 22)

a + b != 27

Por lo tanto no es la respuesta correcta

FASE 3:
POSIBLE SOLUCION

230 = (9 * 22) + (8 * 4)

a + b != 27

Por lo tanto no es la respuesta correcta

RESPUESTA CORRECTA

230 = (9 * 14) + (8 * 13)

a + b = 27

Siendo esta la respuesta correcta


FASE 4. SOLUCION: Hicimos uso del ensayo y error para ver cual de las variables era la correcta y podemos concluir que 14 es la cantidad de cajas con estampas rosadas y 13 la cantidad de cajas con estampas doradas, siendo en total 27 cajas

FASE 1: Encontrar la respuesta de si Matilde debe llevar el abrigo o no.

FASE 2: Para poder resolver este ejercicio se dio el uso de Razonamiento directo y el uso de la siguiente fórmula:
Resultado ºC = (XºF-32) * 5/9

FASE 3: (20 °C * 9/5) + 32 = 68 °F
             (35 °C * 9/5) + 32 = 95 °F


FASE 4. SOLUCION: Simplemente con la palabra VERANO, podemos inferir que no va a hacer frio, además de que con la formula tenemos como resultado que el mínimo es de 20º Centigrados y son 68º Farenheit, por lo tanto no sera necesario el uso del abrigo.
